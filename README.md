## 📌 Sobre

API REST desenvolvida como trabalho final do curso de **Ciência da Computação** na **UNICENTRO**, utilizando **TypeScript**, **Node.js**, **PostgreSQL** e **Docker**.

---

## 🏗️ Arquitetura
```
📁 TrabalhoFinal/
├── 📁 src/
│   ├── 📁 controllers/   # Lógica das rotas
│   ├── 📁 models/        # Modelos do banco de dados
│   ├── 📁 routes/        # Definição das rotas
│   └── 📁 views/         # Templates Handlebars
├── docker-compose.yml
├── package.json
└── tsconfig.json
```
